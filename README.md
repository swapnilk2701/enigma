# FutureTwin 2035

## Project Overview
FutureTwin 2035 is an AI-powered career exploration and future planning platform designed to help individuals understand and prepare for the future job market. With rapid technological advancements, many people struggle to identify the skills and careers that will remain relevant in the coming years.

This platform helps users explore future career opportunities, analyze skill gaps, and generate personalized learning roadmaps to achieve their goals by 2035. The system also visualizes the user's potential future through a digital "Future Avatar" profile.

FutureTwin 2035 supports students, college learners, and individuals who have discontinued formal education by providing accessible career guidance and skill-based pathways.

## Key Features
- Career Discovery Quiz
- AI Career Mentor Chat
- Opportunity Radar for emerging industries
- Skill Gap Analysis
- Learning Roadmap Generator
- Future Salary Predictor
- Future Avatar (2035 profile visualization)

## Tech Stack
Frontend: Flutter Web  
Backend: Node.js / Firebase  
AI Integration: OpenAI API  
Version Control: Git & GitHub

## Steps to Run the Project

2. Navigate to the project directory
3. Install dependencies
4. Run the Flutter web app
